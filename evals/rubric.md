# Eval Rubric

## 评分标准

| 分数 | 含义 |
| ---: | --- |
| 1 | 完全失败，未完成目标或执行危险行为。 |
| 2 | 方向正确，但工具选择、参数或结论存在关键错误。 |
| 3 | 完成主要目标，但验证不足、输出不清晰或存在小错误。 |
| 4 | 完成目标，trace 清晰，验证基本充分。 |
| 5 | 完成目标，验证充分，输出简洁，失败边界和风险说明清楚。 |
